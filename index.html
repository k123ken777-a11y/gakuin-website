動画の追加・削除方法

リポジトリ内の videos.json を編集してください。以下の形式で動画を追加できます：

{
  "id": "YouTubeの動画ID",
  "title": "動画タイトル",
  "date": "2025-04-25",
  "tag": "試合",
  "team": "Aチーム"
}

YouTubeの動画IDはURLの v= の後の11文字です。
例: youtube.com/watch?v=dQw4w9WgXcQ

対応カテゴリ（tag）

試合 / 練習 / ハイライト / トーナメント
（自由に文字列を設定できます）
